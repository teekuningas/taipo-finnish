## ğŸ¹ Chord Reference

**Legend:** `T`=Top, `B`=Bottom, `P`=Pinky, `R`=Ring, `M`=Mid, `I`=Index

### 1. Single Keys (Base)

| Finger | TOP | BOTTOM |
| :--- | :--- | :--- |
| **Pinky** | **r** | **a** |
| **Ring** | **s** | **o** |
| **Middle** | **n** | **t** |
| **Index** | **i** | **e** |

### 2. Same-Row Chords

| Fingers | TOP ROW | BOTTOM ROW |
| :--- | :--- | :--- |
| **I + M** | **y** | **h** |
| **I + R** | **f** | **c** |
| **I + P** | **g** | **d** |
| **M + R** | **p** | **u** |
| **M + P** | **z** | **q** |
| **R + P** | **b** | **l** |

### 3. Cross-Row Chords (Diagonals)

These use one finger from Top and one from Bottom.

| Fingers | Chord | Result |
| :--- | :--- | :--- |
| **P (Top) + I (Bot)** | `TP + BI` | **m** |
| **P (Top) + M (Bot)** | `TP + BM` | **x** |
| **P (Top) + R (Bot)** | `TP + BR` | **Ã¶** |
| **R (Top) + I (Bot)** | `TR + BI` | **v** |
| **R (Top) + M (Bot)** | `TR + BM` | **-** |
| **R (Top) + P (Bot)** | `TR + BP` | **Ã¤** |
| **M (Top) + I (Bot)** | `TM + BI` | **,** |
| **M (Top) + R (Bot)** | `TM + BR` | **+** |
| **M (Top) + P (Bot)** | `TM + BP` | **j** |
| **I (Top) + M (Bot)** | `TI + BM` | **_** |
| **I (Top) + R (Bot)** | `TI + BR` | **k** |
| **I (Top) + P (Bot)** | `TI + BP` | **w** |

### 4. Vertical Chords (Modifiers)

Press Top + Bottom of the *same finger* simultaneously. Sticky (one-shot).

| Finger | Modifier |
| :--- | :--- |
| **Index** (Top+Bot) | **Shift** |
| **Middle** (Top+Bot) | **Ctrl** |
| **Ring** (Top+Bot) | **Alt** |
| **Pinky** (Top+Bot) | **Cmd** |
